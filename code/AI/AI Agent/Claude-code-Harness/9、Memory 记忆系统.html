Memory и®°еҝҶзі»з»ҹ

ж•ҙдҪ“дәӨдә’жөҒзЁӢ

жЁЎеһӢйңҖиҰҒеӯҳеӮЁи®°еҝҶ вҶ’ и°ғз”Ё save_memory(name, body, type, description)
       вҶ“
MemoryManager.save() вҶ’ еҶҷе…Ҙ {name}.md ж–Үд»¶пјҲеҗ« frontmatterпјү вҶ’ йҮҚе»ә MEMORY.md зҙўеј•

жЁЎеһӢйңҖиҰҒжҹҘжүҫи®°еҝҶ вҶ’ и°ғз”Ё search_memory(keyword) / list_memories()
       вҶ“
MemoryManager._scan_files() вҶ’ йҒҚеҺҶ memory_dir/*.md вҶ’ и§Јжһҗ frontmatter вҶ’ иҝ”еӣһ MemoryEntry еҲ—иЎЁ

agent еҗҜеҠЁ / жһ„е»әзі»з»ҹжҸҗзӨәиҜҚж—¶
       вҶ“
MemoryManager.get_memory_context_prompt() вҶ’ иҜ»еҸ–жүҖжңүи®°еҝҶ вҶ’ жӢјжҺҘдёә Markdown ж–Үжң¬ вҶ’ жіЁе…Ҙзі»з»ҹжҸҗзӨәиҜҚ

е®һзҺ°жҖқи·Ҝ

и®°еҝҶзі»з»ҹд»Ҙж–Үд»¶еҚіи®°еҝҶдёәж ёеҝғи®ҫи®ЎпјҡжҜҸжқЎи®°еҝҶжҳҜдёҖдёӘзӢ¬з«Ӣзҡ„ .md ж–Үд»¶пјҢMEMORY.md дҪңдёәзҙўеј•ж–Үд»¶гҖӮ

ж–Үд»¶иҒҢиҙЈ
src/memory/manager.pyи®°еҝҶеӯҳеӮЁгҖҒиҜ»еҸ–гҖҒжҗңзҙўгҖҒеҲ йҷӨ + дёҠдёӢж–ҮжіЁе…Ҙ
src/memory/__init__.pyжЁЎеқ—еҜјеҮә
src/tools/registry.pysave_memory / search_memory / list_memories / delete_memory еӣӣдёӘе·Ҙе…·

еӯҳеӮЁи·Ҝеҫ„дёә ~/.claude/projects/{project_name}-{hash}/memory/пјҢдёҺ Claude Code е…је®№пјҡ

memory/
  MEMORY.md            вҶҗ зҙўеј•пјҲжҜҸиЎҢдёҖжқЎпјҢеҗҜеҠЁж—¶еҠ иҪҪпјү
  user_role.md         вҶҗ з”ЁжҲ·и§’иүІ
  feedback_no_mock.md  вҶҗ з”ЁжҲ·еҸҚйҰҲ
  project_freeze.md    вҶҗ йЎ№зӣ®зәҰе®ҡ
  ref_grafana.md       вҶҗ еӨ–йғЁиө„жәҗ

жЁЎеһӢйҖҡиҝҮеӣӣдёӘе·Ҙе…·пјҲsave_memoryгҖҒsearch_memoryгҖҒlist_memoriesгҖҒdelete_memoryпјүж“ҚдҪңи®°еҝҶгҖӮжҜҸж¬Ў save жҲ– delete еҗҺиҮӘеҠЁйҮҚе»ә MEMORY.md зҙўеј•гҖӮ

ж ёеҝғй—®йўҳ

Q1: Memory еӯҳд»Җд№Ҳпјҹ

и®°еҝҶеҲҶдёәеӣӣз§Қзұ»еһӢпјҢжҜҸз§ҚеҜ№еә”дёҚеҗҢзҡ„жҢҒд№…еҢ–еҶ…е®№пјҡ

зұ»еһӢеӯҳд»Җд№ҲзӨәдҫӢ
userз”ЁжҲ·и§’иүІгҖҒеҒҸеҘҪгҖҒзҹҘиҜҶиғҢжҷҜ"з”ЁжҲ·жҳҜеҗҺз«Ҝе·ҘзЁӢеёҲпјҢзҶҹжӮү Python е’Ң Go"
feedbackз”ЁжҲ·жҳҺзЎ®зә жӯЈжҲ–зЎ®и®Өзҡ„ең°ж–№"д»ҘеҗҺйҒҮеҲ°иҝҷз§Қжғ…еҶөиҰҒе…ҲеҒҡ X"гҖҒ"дёҚиҰҒиҝҷж ·ж”№"
projectдёҚе®№жҳ“д»Һд»Јз ҒзӣҙжҺҘзңӢеҮәжқҘзҡ„йЎ№зӣ®зәҰе®ҡжҲ–иғҢжҷҜ"жң¬е‘ЁдёүеүҚйңҖиҰҒе®ҢжҲҗз§»еҠЁз«ҜеҸ‘еёғеҲҶж”Ҝ"гҖҒ"и®ӨиҜҒдёӯй—ҙд»¶йҮҚеҶҷжҳҜеӣ дёәеҗҲи§„иҰҒжұӮ"
referenceеӨ–йғЁиө„жәҗжҢҮй’Ҳ"Bug еңЁ Linear йЎ№зӣ® INGEST дёӯиҝҪиёӘ"гҖҒ"Grafana йқўжқҝеңЁ grafana.internal/d/api-latency"

ж ёеҝғеҺҹеҲҷпјҡеҸӘеӯҳ"и·ЁдјҡиҜқд»Қжңүд»·еҖј"дё”"ж— жі•д»ҺеҪ“еүҚд»Јз ҒзҠ¶жҖҒжҺЁеҜјеҮәжқҘ"зҡ„дҝЎжҒҜгҖӮ

Q2: Memory д»Җд№ҲдёҖе®ҡдёҚиғҪеӯҳпјҹ

д»ҘдёӢдҝЎжҒҜдёҚеә”еҶҷе…Ҙи®°еҝҶпјҡ

дёҚеә”еӯҳзҡ„еҶ…е®№еҺҹеӣ 
ж–Үд»¶з»“жһ„гҖҒеҮҪж•°зӯҫеҗҚгҖҒзӣ®еҪ•еёғеұҖеҸҜд»ҘзӣҙжҺҘиҜ»еҸ–д»Јз ҒиҺ·еҫ—
еҪ“еүҚд»»еҠЎиҝӣеәҰеұһдәҺ Plan / Task зі»з»ҹзҡ„иҒҢиҙЈ
дёҙж—¶еҲҶж”ҜеҗҚгҖҒеҪ“еүҚ PR еҸ·еҫҲеҝ«иҝҮж—¶пјҢи®°еҝҶдјҡ stale
дҝ® bug зҡ„е…·дҪ“д»Јз Ғз»ҶиҠӮgit жҸҗдәӨи®°еҪ•е·Іжңү
еҜҶй’ҘгҖҒеҜҶз ҒгҖҒеҮӯиҜҒе®үе…ЁйЈҺйҷ©
еҜ№иҜқдёӯзҡ„дёҙж—¶дёҠдёӢж–ҮдјҡиҜқз»“жқҹеҗҺж— д»·еҖј

еҲӨж–ӯж ҮеҮҶпјҡ еҰӮжһңиҝҷжқЎдҝЎжҒҜеҸҜд»Ҙд»Һд»Јз ҒгҖҒgit еҺҶеҸІгҖҒжҲ–еҪ“еүҚд»»еҠЎзҠ¶жҖҒдёӯйҮҚж–°иҺ·еҫ—пјҢе°ұдёҚеә”иҜҘеӯҳеҲ°и®°еҝҶйҮҢгҖӮ

Q3: жЁЎеһӢеҰӮдҪ•еӯҳеӮЁ Memoryпјҹ

жЁЎеһӢйҖҡиҝҮеӣӣдёӘе·Ҙе…·дёҺи®°еҝҶзі»з»ҹдәӨдә’пјҢдёҚйңҖиҰҒзӣҙжҺҘж“ҚдҪңж–Үд»¶пјҡ

дҝқеӯҳи®°еҝҶ вҖ” save_memory

жЁЎеһӢи°ғз”Ё save_memory(name="user_role", body="еҗҺз«Ҝе·ҘзЁӢеёҲ, Python + Go", type="user", description="з”ЁжҲ·и§’иүІ")
       вҶ“
registry.py вҶ’ _save_memory() вҶ’ MemoryManager.save()
       вҶ“
1. ж №жҚ® name з”ҹжҲҗе®үе…Ёж–Үд»¶еҗҚпјҲзү№ж®Ҡеӯ—з¬ҰжӣҝжҚўдёә _пјү
2. еҶҷе…Ҙ {name}.mdпјҢж јејҸпјҡfrontmatter + body
3. _rebuild_index() жү«жҸҸжүҖжңү .md ж–Үд»¶ вҶ’ йҮҚеҶҷ MEMORY.md

з”ҹжҲҗзҡ„ж–Үд»¶еҶ…е®№пјҡ

---
name: user_role
description: з”ЁжҲ·и§’иүІ
type: user
---

еҗҺз«Ҝе·ҘзЁӢеёҲ, Python + Go

жҗңзҙўи®°еҝҶ вҖ” search_memory

жЁЎеһӢи°ғз”Ё search_memory(keyword="mock")
       вҶ“
MemoryManager.search() вҶ’ _scan_files() вҶ’ йҒҚеҺҶ name + description + body еҢ№й…Қе…ій”®иҜҚ
       вҶ“
иҝ”еӣһеҢ№й…ҚжқЎзӣ®еҲ—иЎЁпјҲж јејҸпјҡ"[type] name: body еүҚ 300 еӯ—з¬Ұ"пјү

еҲ—еҮәжүҖжңүи®°еҝҶ вҖ” list_memories

жЁЎеһӢи°ғз”Ё list_memories()
       вҶ“
MemoryManager.load_all() вҶ’ _scan_files() вҶ’ иҝ”еӣһжүҖжңүжқЎзӣ®зҡ„ name + type + description

еҲ йҷӨи®°еҝҶ вҖ” delete_memory

жЁЎеһӢи°ғз”Ё delete_memory(name="old_memory")
       вҶ“
MemoryManager.delete() вҶ’ еҲ йҷӨж–Үд»¶ вҶ’ _rebuild_index()

еҗҢеҗҚиҰҶзӣ–пјҡ еҰӮжһң save_memory зҡ„ name дёҺе·Іжңүи®°еҝҶзӣёеҗҢпјҢзӣҙжҺҘиҰҶзӣ–ж—§ж–Үд»¶пјҲе…ҲеҶҷж–°еҶ…е®№пјҢеҶҚйҮҚе»әзҙўеј•пјүгҖӮ

Q4: Memory ж–Үд»¶дёӯеҢ…еҗ«д»Җд№Ҳпјҹ

жҜҸжқЎи®°еҝҶж–Үд»¶жҳҜдёҖдёӘеёҰ frontmatter зҡ„ Markdown ж–Үд»¶пјҡ

---
name: feedback_no_mock
description: йӣҶжҲҗжөӢиҜ•еҝ…йЎ»з”Ёзңҹе®һж•°жҚ®еә“пјҢдёҚз”Ё mock
type: feedback
---

з”ЁжҲ·иҰҒжұӮйӣҶжҲҗжөӢиҜ•дёҚдҪҝз”Ё mock ж•°жҚ®еә“гҖӮеҺҹеӣ пјҡд№ӢеүҚеҮәзҺ°иҝҮ mock е’Ңз”ҹдә§ж•°жҚ®дёҚдёҖиҮҙеҜјиҮҙиҝҒз§»еӨұиҙҘзҡ„й—®йўҳгҖӮеә”з”Ёж–№ејҸпјҡжүҖжңүж•°жҚ®еә“зӣёе…іжөӢиҜ•дҪҝз”Ё testcontainers жҲ–зңҹе®һжөӢиҜ•еә“гҖӮ

frontmatter еӯ—ж®өпјҡ

еӯ—ж®өеҝ…йңҖиҜҙжҳҺ
nameжҳҜи®°еҝҶзҡ„е”ҜдёҖж ҮиҜҶпјҢд№ҹз”ЁдҪңж–Үд»¶еҗҚ
descriptionжҳҜдёҖиЎҢж‘ҳиҰҒпјҢжҳҫзӨәеңЁ MEMORY.md зҙўеј•дёӯ
typeжҳҜи®°еҝҶзұ»еһӢпјҲuser / feedback / project / referenceпјү

MEMORY.md зҙўеј•ж–Үд»¶пјҡ

- [feedback_no_mock](feedback_no_mock.md) вҖ” йӣҶжҲҗжөӢиҜ•еҝ…йЎ»з”Ёзңҹе®һж•°жҚ®еә“
- [project_freeze](project_freeze.md) вҖ” 3жңҲ5ж—ҘеҗҺеҶ»з»“йқһе…ій”®еҗҲе№¶
- [ref_grafana](ref_grafana.md) вҖ” API е»¶иҝҹзӣ‘жҺ§йқўжқҝ

жҜҸжқЎдёҖиЎҢпјҢдёҚи¶…иҝҮ 150 еӯ—з¬ҰгҖӮagent еҗҜеҠЁж—¶е…ҲиҜ»еҸ–зҙўеј•пјҢж №жҚ® description еҲӨж–ӯжҳҜеҗҰйңҖиҰҒиҜ»еҸ–е…·дҪ“ж–Үд»¶еҶ…е®№пјҢйҒҝе…Қж— е·®еҲ«еҠ иҪҪжүҖжңүи®°еҝҶгҖӮ

жіЁе…Ҙзі»з»ҹжҸҗзӨәиҜҚж—¶пјҡ get_memory_context_prompt(max_chars=4000) жҢүйЎәеәҸжӢјжҺҘи®°еҝҶжӯЈж–ҮпјҢи¶…иҝҮ max_chars еӯ—з¬Ұж—¶жҲӘж–ӯпјҢиҫ“еҮәж јејҸпјҡ

Q5: и®°еҝҶе…·дҪ“жҳҜжҖҺд№Ҳжҗңзҙўзҡ„пјҹ

и®°еҝҶжҗңзҙўжңүдёӨжқЎи·Ҝеҫ„пјҡеҗҜеҠЁж—¶зҡ„зҙўеј•еҠ иҪҪпјҲиў«еҠЁпјүе’ҢжЁЎеһӢдё»еҠЁи°ғз”Ё search_memoryпјҲдё»еҠЁпјүгҖӮ

и·Ҝеҫ„дёҖпјҡеҗҜеҠЁж—¶ вҖ” зҙўеј•еҝ«йҖҹжөҸи§Ҳ

agent еҗҜеҠЁ
      вҶ“
MemoryManager.get_memory_context_prompt(max_chars=4000)
      вҶ“
load_all() вҶ’ _scan_files() вҶ’ йҒҚеҺҶ memory_dir/*.mdпјҲжҺ’йҷӨ MEMORY.mdпјү
      вҶ“
еҜ№жҜҸдёӘ .md ж–Үд»¶пјҡ
    1. иҜ»еҸ–ж–Үд»¶е…Ёж–Ү
    2. з”ЁжӯЈеҲҷи§Јжһҗ frontmatterпјҲname / description / typeпјү
    3. жҸҗеҸ– body жӯЈж–Ү
      вҶ“
жҢүж–Үд»¶жҺ’еәҸйЎәеәҸжӢјжҺҘпјҢи¶…иҝҮ max_chars(4000) еӯ—з¬Ұж—¶жҲӘж–ӯ
      вҶ“
жіЁе…Ҙзі»з»ҹжҸҗзӨәиҜҚпјҢж јејҸпјҡ
    "## Memory (persisted across sessions)\n\n## name [type]\nbody\n"

е…ій”®з»ҶиҠӮпјҡ

  • жү«жҸҸиҢғеӣҙпјҡmemory_dir дёӢжүҖжңү .md ж–Үд»¶пјҢжҺ’йҷӨ MEMORY.md зҙўеј•ж–Үд»¶жң¬иә«
  • frontmatter и§ЈжһҗпјҡжӯЈеҲҷ ^---\s*\n(.*?)\n---\s*\n(.*) еҢ№й…Қ YAML-like еӨҙпјҢеҶҚйҖҗиЎҢ key: value и§Јжһҗ
  • жҺ’еәҸпјҡжҢүж–Үд»¶еҗҚ sorted() еӯ—жҜҚеәҸ
  • жҲӘж–ӯпјҡи¶…иҝҮ max_chars ж—¶зӣҙжҺҘеҒңжӯўжӢјжҺҘпјҢеҗҺз»ӯи®°еҝҶдёҚжіЁе…Ҙ

MEMORY.md зҡ„дҪңз”Ёпјҡ зҙўеј•ж–Үд»¶еңЁеҗҜеҠЁж—¶д№ҹдјҡиў«еҠ иҪҪеҲ°дёҠдёӢж–ҮпјҲзі»з»ҹжҸҗзӨәиҜҚиҜҙжҳҺдёӯеј•з”ЁпјүпјҢи¶…иҝҮ 200 иЎҢдјҡиў«жҲӘж–ӯгҖӮе®ғи®© agent еңЁдёҚжү“ејҖе…·дҪ“ж–Үд»¶зҡ„жғ…еҶөдёӢпјҢйҖҡиҝҮ description еҝ«йҖҹеҲӨж–ӯе“Әдәӣи®°еҝҶеҖјеҫ—иҜ»еҸ–гҖӮ

и·Ҝеҫ„дәҢпјҡжЁЎеһӢдё»еҠЁжҗңзҙў вҖ” search_memory(keyword)

жЁЎеһӢи°ғз”Ё search_memory(keyword="mock")
      вҶ“
MemoryManager.search(keyword)
      вҶ“
1. _scan_files() вҶ’ жү«жҸҸжүҖжңү .md ж–Үд»¶пјҢи§Јжһҗдёә MemoryEntry
2. keyword_lower = "mock"пјҲиҪ¬дёәе°ҸеҶҷпјү
3. йҖҗжқЎеҢ№й…ҚпјҲеӯҗеӯ—з¬ҰдёІеҢ…еҗ«пјҢдёҚеҢәеҲҶеӨ§е°ҸеҶҷпјүпјҡ
    keyword in entry.name.lower()          вҶҗ еҢ№й…ҚеҗҚз§°
    OR keyword in entry.description.lower() вҶҗ еҢ№й…ҚжҸҸиҝ°
    OR keyword in entry.body.lower()        вҶҗ еҢ№й…ҚжӯЈж–Ү
4. иҝ”еӣһжүҖжңүеҢ№й…Қзҡ„ MemoryEntry еҲ—иЎЁ

жҗңзҙўзү№зӮ№пјҡ

зү№жҖ§е®һзҺ°ж–№ејҸ
еҢ№й…Қз®—жі•з®ҖеҚ•зҡ„еӯҗеӯ—з¬ҰдёІеҢ…еҗ«пјҲin ж“ҚдҪңз¬ҰпјүпјҢйқһжЁЎзіҠеҢ№й…Қ/жӯЈеҲҷ/еҲҶиҜҚ
еӨ§е°ҸеҶҷдёҚж•Ҹж„ҹпјҲжҗңзҙўеүҚз»ҹдёҖиҪ¬е°ҸеҶҷпјү
жҗңзҙўиҢғеӣҙname + description + body дёүдёӘеӯ—ж®ө
еӨҡе…ій”®иҜҚдёҚж”ҜжҢҒпјҢжҜҸж¬ЎеҸӘиғҪжҗңдёҖдёӘе…ій”®иҜҚпјҲжЁЎеһӢеҸҜд»ҘеӨҡж¬Ўи°ғз”Ёпјү
жҺ’еәҸжҢүж–Үд»¶еҗҚжҺ’еәҸпјҢдёҚжҢүзӣёе…іжҖ§
ж–Үд»¶йҒҚеҺҶжҜҸж¬Ўи°ғз”Ё search() йғҪйҮҚж–°жү«жҸҸзЈҒзӣҳпјҲ_scan_files()пјүпјҢж— еҶ…еӯҳзј“еӯҳ

дёӨжқЎи·Ҝеҫ„зҡ„еҜ№жҜ”пјҡ
